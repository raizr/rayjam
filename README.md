### Description
 Game for Raylib Slo-Jam. Developed with raylib for web and win builds

### Controls
Keyboard:
 - W - thrust
 - A/D - Left/Right control
 - S - Break
 - L Shift - Boost
 - Space - Shoot
 - Esc - Menu
